# Initial Concept
A TeachBook template for creating and hosting interactive Jupyter books without extensive knowledge of Git or web servers, designed to serve University Students, Professional Learners, and Practitioners using a Problem-Based Learning approach.

# Product Guide

* The book is focused on covering the material in third_party/S-5A_Ed1.0.2.md for the IHO Cat A hydrographer certification.
* All answers in the book should have references.
* Python should be the preferred programming language.
* The students should be assumed to be on Ubuntu Linux, but with access to MS Windows on a remote virtual machine.
* The answers should prefer using open source software whenever possible, but list commercial alternatives.
* Do not use any AGPL programs or similarly licensed software.
* Database examples should try to provide alternative examples using SQLite and DuckDB.
* Text and generated data in the book should be under the Creative Commons CC-0 license. Code should be under the Apache-2.0 license. The original book template is CC-BY-4.0.
* Reference SPDX whenever relevant.
* When possible, provide unit tests for code examples.
* The project to aim for correctness and completeness.
* Assume the reader does not know acronyms and terms used in the field and provide a glossary, links to primary references, wikipedia/wiktionary, and https://gcmd.earthdata.nasa.gov/KeywordViewer/scheme/all.
